# GCTW AI Chatbot for Giri Cooling Tower Works

# AI-powered chatbot built with [Botpress](https://botpress.com/?utm_source=chatgpt.com) for Giri Cooling Tower Works to automate lead qualification, quotation requests, technical support, and consultation scheduling for cooling tower products and services across India.***

## 🚀 Project Overview

# This project is a production-ready conversational AI assistant designed for industrial B2B sales and service automation.The chatbot helps customers:* Inquire about cooling towers, spare parts, and maintenance services
* Request quotations
* Troubleshoot common cooling tower issues
* Book free consultation calls and site inspections
* Get answers from a technical knowledge base
* Connect with the technical sales teamThe chatbot is configured with:* **Basic Prompt** for conversation logic
* **Knowledge Base** for FAQs and technical answers
* **Tool Instructions** for Google Calendar scheduling
* **Google Calendar Integration**
* **WhatsApp and Website Chat Support*****

## 🤖 Chatbot Features

### Lead Qualification

# The bot collects:* Product or service required
* Cooling tower capacity (TR)
* City and company name

### Quotation Requests

# Captures customer details and forwards them to the sales team.

### Technical Support

# Answers common issues such as:* High outlet temperature
* Fan vibration
* Water leakage
* Fill replacement

### Appointment Scheduling

# Uses Google Calendar to:* Check availability
* Offer three appointment options
* Create consultation events

### Knowledge Base Search

# Answers frequently asked questions about:* Products
* Services
* AMC contracts
* Spare parts
* Delivery timelines

### Multi-Language Support

# Responds in:* English
* Hindi
* Telugu
* Tamil
* Kannada***
